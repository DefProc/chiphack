Chip Hack

This repository (and its companion wiki) contains tutorial code and documentation for the Chip Hack Introduction to FPGA Programming event, a 2 day workshop originally run on the weekend of Saturday 20th and Sunday 21st April 2013 at The Centre for Creative Collaboration in London, UK.

The materials will continue to be updated and a second Chip Hack event is tentatively planned for October 2013 in Cambridge, UK.

All are encouraged to use and contribute to these materials, issues should be logged using the issue tracker, and there is a Google group for discussion.
